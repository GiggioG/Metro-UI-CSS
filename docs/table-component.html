Table component

Create interactive table with sorting, filtering and pagination features.

About

New in 4.2.8, updated in 4.2.23

If you need to create interactive table with filtering, searching and pagination features, you can use Metro 4 table component. To create table, add attribute data-role="table" to your HTML table element and setup, if need, additional options with special attributes. You can create table and for your own HTML table with predefined data and for the table with remote data definition.

An important condition is the creation of sections thead and tbody. 


                    <table class="table" data-role="table">
                    ...
                    </table>
