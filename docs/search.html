Input

Enhancement for standard HTML input[type=text] to the search input.

About

In Metro 4, you can use enhancement version of input element for creating search input control. To define it, add attribute data-role="search" to HTML input element. You can assign role input to any text inputs. 


                    <input type="text" data-role="search">

Prepend data

You can add prepend data to input field with attribute data-prepend="...". 


                    <input type="text" data-role="search" data-prepend="User name: ">

Search and clear buttons

When Metro 4 create input component, him add to element two special buttons: clear-button and search-button.

Clear button

This button clear current value of input. You can disable clear-button with attribute data-clear-button="false". You can change default icon for this button. To set your icon, use attribute data-clear-button-icon="...".

Search button

When user press this button:

  1. If You define attribute data-search-button-click="submit", Metro 4 execute your function, defined with attribute data-on-search-button-click="..."
  2. If you define attribute data-search-button-click with other value, Metro 4 submit a form, where search input is defined

You can disable search-button with attribute data-search-button="false". You can change default icon for this button. To set your icon, use attribute data-search-button-icon="...".

Define search button click

Custom func

Form submit func


                    <script>
                        function mySubmitSearch(val){
                            alert('Custom submit function');
                        }
                    </script>
                    <form action="javascript:" onsubmit="alert('Form submit with self function');">
                        <p>Custom func</p>
                        <input type="text" data-role="search" data-search-button-click="custom" data-on-search-button-click="mySubmitSearch">

                        <p>Form submit func</p>
                        <input type="text" data-role="search">
                    </form>

Custom buttons

You can create custom button for input control. To add custom button, first-off - create array with your buttons. This array must be created before input component initiated. 


                    <script>
                        var customButtons = [
                            {
                                html: "<span class='mif-user'></span>",
                                cls: "alert",
                                onclick: "alert('You press user button')"
                            },
                            {
                                html: "<span class='mif-cog'></span>",
                                cls: "warning",
                                onclick: "alert('You press cog button')"
                            }
                        ]
                    </script>

                    <input type="text" data-role="search" data-custom-buttons="customButtons">

Options

Option Data-* Default Desc
size data-size Set specific size for input
prepend data-prepend Add prepend label
clearButton data-clear-button true Add clear button to input
clearButtonIcon data-clear-button-icon <span class='default-icon-cross'> Icon for clear button
searchButton data-search-button true Add search button to input
searchButtonIcon data-search-button-icon <span class='default-icon-search'> Icon for search button
customButtons data-custom-buttons Array name with custom buttons
searchButtonClick data-search-button-click submit Behavior of the serach button when user clicked on it. Value must be submit or custom
onSearchButtonClick data-on-search-button-click Metro.noop Function for custom search method

Events

Event Data-* Context
onSearchButtonClick(val) data-on-search-button-click input
onSearchCreate(el) data-on-search-create input

Methods

You can use methods to interact with input component: enable(), disable(), toggleState(). 


                    var input = $(el).data('input');
                    input.toggleState();

Customize

You can customize your input with special attributes:

Option Data-* Desc
clsSearch data-cls-search Additional classes for input control.
clsInput data-cls-input Additional classes for input element.
clsPrepend data-cls-prepend Additional classes for input prepend data.
clsClearButton data-cls-clear-button Additional classes for input clear button.
clsSearchButton data-cls-search-button Additional classes for input search button.
clsCustomButton data-cls-custom-button Additional classes for input custom button.
