What is it?

Metro 4 is an open source toolkit for developing with HTML, CSS, and JS. Quickly prototype your ideas or build your entire app with responsive grid system, extensive prebuilt components, and powerful plugins built on jQuery.

Build responsive, mobile-first projects on the web with the first front-end component library in Metro Style.

All the Features You Need
With Metro 4 you get:

Base styles

Metro 4 includes various styles for changing the appearance of html elements.

70+ Components

More than 70 components are available to you to solve almost all tasks.

Tools and Routines

Various built-in tool functions and classes will increase your productivity.

Issues

Rapid response to detected bugs and their elimination.
